Overlord Bot

What is Overlord Bot?

Overlord Bot is a personal automated assistant that manages workflows, monitors GitHub repositories, and handles email and chat messages on behalf of its operator. It runs as a scheduled job system, processing tasks automatically according to predefined rules.

Google APIs Used

Overlord Bot accesses the following Google APIs on behalf of the authorized Google account:

  • Gmail API -- read, send, and manage emails for the authorized account
  • Google Docs API -- create, read, and modify documents
  • Google Slides API -- create and modify presentations
  • Google Sheets API -- create, read, and modify spreadsheets
  • Google Calendar API -- read and manage calendar events

All API access is limited to the single authorized Google account. The bot does not access data belonging to any other users.
